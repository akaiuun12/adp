# Codes
- [ADP_Python](https://github.com/ADPclass/ADP_book_ver01)

각 책에 해당하는 코드를 첨부한다. Python책은 코드를 GPL 라이선스로 풀었지만 R책은 데이터에듀 홈페이지에서 구매인증 후 다운받도록 하고 있다.
따라서 이 레포지토리도 GPL 라이선스를 따르며 R책의 내용은 다루지만 R책 코드는 포함하지 않도록 한다.
